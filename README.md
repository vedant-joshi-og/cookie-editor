# Cookie Editor

Mess with cookies. Fix logins. Test stuff. Break things (on purpose).  
**Cookie Editor** lets you view, edit, add, delete cookies right from your browser — fast, simple, no drama.

Built for devs, testers, and curious internet goblins.

---

[Firefox](https://addons.mozilla.org/en-US/firefox/addon/cookie-editor-popup/)

---

## What it does

- View all cookies for current site  
- Edit cookie values on the fly  
- Add new cookies  
- Delete annoying ones  
- Instant apply (no reload rage)

---

## Setup

1. Download or clone this repo  
2. Open your browser → Extensions  
3. Turn on **Developer Mode**  
4. Click **Load Unpacked**  
5. Select the project folder  
6. Done.

## How to use

1. Open any website  
2. Click the **Cookie Editor** icon  
3. Do your cookie magic  
4. Refresh if needed  
5. Profit

---

## Notes

- Changes affect the current site only  
- Clearing cookies may log you out (obviously)  
- Don’t blame the tool if you nuke your session
